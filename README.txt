# ✅ Task Manager (Python CLI)

A simple yet effective **Command-Line Task Manager** written in Python, using **JSON** for persistent storage.  
Easily add, delete, mark as done, and list tasks directly in your terminal.

---

## 💡 Features

- 🔸 Add new tasks
- 🔸 Delete tasks by ID
- 🔸 Mark tasks as done
- 🔸 Show task list with status (✅ or ❌)
- 🔸 Data saved in a local `task.json` file
